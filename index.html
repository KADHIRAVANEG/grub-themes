## Install

Clone the repo and run the interactive installer. The script backs up /etc/default/grub automatically.

git clone <your-repo-url> gorgeous-grub
cd gorgeous-grub
chmod +x install.sh
sudo ./install.sh                # interactive dropdown
sudo ./install.sh miku           # pick directly by slug
sudo ./install.sh -l             # list themes
sudo ./install.sh -u             # restore previous config
